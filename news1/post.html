Problems look mighty small from 150 miles up

Posted by Start Bootstrap on August 24, 2018

אם אתם מכירים גם את פייתון וגם את דוט נט ויודעים את היתרונות שיש לכל טכנולוגיה ,אז מדוע שלא נחשוב לאחד בן השתיים אולי להריץ קוד פייתון וקוד c# אחד לצד השני.

הסיפור הזה מאז ומעולם היה מורכב והיו דרכים מגוונות כדי לבצע את זה וכאן נסקור את חלקן.

דרך אחת באמצעות למעשה להריץ תהליך בתוך אפליקציה דוטנט שלנו ,שהתהליך למעשה הוא אפליקציה שמסוגלת להריץ את קוד פייתון במצב הפשוט הכוונה היא לתהליך cmd שישתמש בפייתון שמותקן אצלנו על המחשב ניתן לו כארגומנט את הקובץ וכך הוא יבצע הרצה של קובץ הפייתון.

באותה גישה ניתן ליצור קובץ הרצה מיקוד פייתון באמצעות pyinstallerו אז להריץ את קובץ ההרצה באמצעות מחלקת process .

ישנן מספר ספריות שנבנו על בסיס הכלים האלו

בנוסף גישה נוספת שהייתה ליצור למעשה api ואז מתוך קוד c sharp באמצעות מחלקת httpclient נקרא לנקודת הקצה שמבוססת על פייתון ומחזירה את הערכים שפייתון מייצר.

דוגמאות וספריות הסוגים שהזכרנו כאן – python.net ו iron.python ועוד.

הספריה החדשה שכיום אנחנו מציגים אותה פשוט מייצרת את כל התהליך מאחורי הקלעים ומבצעת את הריאקציה המושלמת בין פייתון וc# כך שנוכל ממש לקרוא פונקציות פייתון מתוך קוד c# והסיפור האמיתי שזה פשוט עובד בצורה מאוד מאוד פשוטה בדרך הרגילה שהתרגלתם לכתוב קוד פייתון ובדרך הרגילה שהתרגלתם לכתוב קוד c# ניתן לעבוד במקביל בין השתיים.
